## Default handle
The default handle is `@coderabbitai` but sometimes it could be `@coderabbit`

## Commands
- `@coderabbitai review` is incremental only
- `@coderabbitai full review`
- `@coderabbitai resolve` marks all previous comments as resolved

If you find there're a lot of outdated unresolved threads and you want to clean them all and review as a whole again:
```text
@coderabbitai resolve
@coderabbitai full review
```
